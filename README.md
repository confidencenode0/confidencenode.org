# confidencenode.org

Static site for confidencenode.org.

The `trusteando/` folder is the ConfidenceNode Trusteando node — a Level 1
participation in the Trusteando Protocol. It publishes verifiable facts about
the organization: identity, members, and protocols authored.

## Structure

```
trusteando/
├── identity/               ← organization identity
├── members/                ← founding members
├── protocols/              ← protocols published by this organization
└── [state verifiado]/      ← conformity state
```

## Deployment

GitHub Pages — branch `main`, root `/`.

Configure custom domain `confidencenode.org` in repository settings.
